# folder-bomb
Program that creates lots of directories in a specific directory.

Made just for fun.

# Usage
```
python3 folder-bomb.py NUMBER-OF-FOLDERS-TO-CREATE DIRECTORY-WHERE-TO-CREATE-THEM
```
